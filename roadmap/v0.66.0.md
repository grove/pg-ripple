# v0.66.0 - Streaming and Distributed Reality

> **Full technical details:** [v0.66.0-full.md](v0.66.0-full.md)

**Status: Planned** | **Scope: Very Large**

> v0.66.0 makes the streaming, Arrow, WCOJ, and Citus claims real or explicitly downgraded. It replaces materialized cursors with true streaming, implements signed-ticket Arrow IPC export, wires Citus helpers into SPARQL/storage execution, and makes cyclic join optimization explainable as either planner hints or a real executor.

---

## What is this?

The assessment found several advanced features whose public story exceeded their implementation: Arrow Flight is a JSON stub, SPARQL cursors materialize results, Citus helpers are not wired into translators, and WCOJ is planner guidance rather than a true Leapfrog Triejoin executor. v0.66.0 closes or truth-labels those gaps.

## Headline outcomes

- SPARQL cursor and HTTP streaming memory are bounded by page size, not full result size.
- Arrow Flight returns signed, expiring tickets and streams real Arrow IPC batches over promoted VP tables and `vp_rare`.
- Citus SERVICE pruning, HLL distinct, per-worker BRIN summarise, non-blocking promotion, and RLS propagation are integrated and tested.
- WCOJ explain output accurately reports planner-hint vs executor mode.
- Distributed and streaming fallbacks are visible in `feature_status()`, `sparql_explain`, metrics, and docs.

## Assessment findings covered

- C4/P5/S2: Arrow Flight is a stub, tickets are unsigned, and promoted VP tables are not streamed.
- H1/P4/SC4: WCOJ is planner guidance, session preamble can leak, and docs describe nonexistent plan nodes.
- H2/P2/P3/S4: v0.63 Citus claims are helpers or dead paths, approximate distinct is not query pushdown, promotion is synchronous, and worker RLS propagation lacks runtime tests.
- H3: SPARQL cursor materializes results before returning them.
- Operational gaps: cursor/export memory metrics, Arrow ticket metrics, Citus BRIN/shard lag, and distributed fallback reasons.
- Recommended new features: Explainable Distributed SPARQL and Real Arrow Flight/ADBC Export Path.
