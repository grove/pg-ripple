# v0.83.1 — Assessment 13 Critical/High & Operational Remediation

> Source: PLAN_OVERALL_ASSESSMENT_13.md  
> Status: Planned  
> Scope: Large

## Summary

v0.83.1 closes the ten "must-fix before v1.0.0" findings from Assessment 13.
The headline items are HTTP companion version synchronisation (the companion has
been 6 versions behind the extension since v0.77.0), docker-compose image tag
currency, SECURITY DEFINER inline documentation, CI format-check gate, migration
chain coverage for v0.80–v0.83, and the nested OPTIONAL+EXISTS correctness fix.
Code-quality and DevEx items complete the release: `gucs/registration.rs` split
(2 032 lines), `/health/ready` deep-check, plan cache double-parse elimination, and
`justfile` automation recipes.
