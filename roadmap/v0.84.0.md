# v0.84.0 — Assessment 13 Critical/High & Operational Remediation

> **Full technical details:** [v0.84.0-full.md](v0.84.0-full.md)

**Status: Planned** | **Scope: Large**

> v0.84.0 closes all Critical and High findings from Assessment 13
> (`plans/PLAN_OVERALL_ASSESSMENT_13.md`). The headline items are: HTTP
> companion version sync (6-version drift since v0.77.0), docker-compose image
> tag currency, SECURITY DEFINER inline documentation, CI SQL-injection-check
> gate, migration-chain coverage for v0.80–v0.83, gucs/registration.rs split
> (2,032 lines), /health/ready deep-check, plan cache double-parse elimination,
> and justfile automation recipes.
