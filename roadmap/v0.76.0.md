# v0.76.0 - Assessment 11 Low-Severity Findings and Production Polish

> **Full technical details:** [v0.76.0-full.md](v0.76.0-full.md)

**Status: Planned** | **Scope: Medium**

> v0.76.0 addresses the Low-severity findings and enhancements from Assessment 11: toolchain pinning, policy hash widening, Arrow dependency pinning, benchmark baseline refresh, test count growth, /metrics auth documentation, xact callback safety documentation, log-hook defense-in-depth, and miscellaneous CI verification.

---

## What is this?

The final Assessment 11 remediation release before v1.0.0. Every Low-severity finding and enhancement noted in Assessment 11 is addressed here, closing the coverage gap.

## Headline outcomes

- **TOOLCHAIN-PIN-01**: `rust-toolchain.toml` pins `channel` to a specific stable version (e.g. `1.87.0`) instead of `"stable"`. Dependabot or Renovate configured to update the pin. Closes Low finding: rust-toolchain.toml pins "stable" not specific version.
- **RLS-HASH-01**: RLS policy name generation upgraded from 64-bit xxhash to 128-bit to reduce 4-billion-graph collision probability. Closes Low finding: policy name 64-bit hash collision risk.
- **ARROW-PIN-01**: `pg_ripple_http/Cargo.toml` pins `arrow` to a specific minor version (e.g. `arrow = "55.1"`) instead of just the major. Closes Low finding: arrow dep not pinned to minor.
- **BENCH-REFRESH-01**: `benchmarks/merge_throughput_baselines.json` updated from v0.53.0 baseline to current version. Closes Low finding: merge baselines stale.
- **TEST-GROWTH-01**: New regression tests added across the release train to approach the 300+ target. Current count ~190; this release adds targeted tests to bring the total above 220. Closes Low finding: test count ~190 vs 300+ target.
- **METRICS-AUTH-DOC-01**: `/metrics` endpoint auth model documented in the HTTP companion operations docs. Closes Low finding: /metrics auth undocumented.
- **XACT-SPI-DOC-01**: `src/lib.rs` xact callback comment about `XACT_EVENT_PRE_COMMIT` SPI safety updated with an authoritative PostgreSQL source citation. Closes Low finding: xact callback PRE_COMMIT SPI safety claim.
- **LOG-HOOK-01**: `RegisterEmitLogHook` defense-in-depth scan: verify no raw HMAC keys, connection strings, or secrets are logged in any error path. Document findings. Closes Low finding: no RegisterEmitLogHook for secrets suppression.
- **CLIPPY-VERIFY-01**: `clippy --deny warnings` CI gate re-verified for the current codebase. Closes Low finding: clippy gate not re-verified.
- **LLM-KGE-STATUS-01**: Confirm `src/llm/` and `src/kge.rs` are represented in `feature_status()` (per v0.73.0 FEATURE-STATUS-02). Closes Low finding: src/llm/ and src/kge.rs not in feature_status (verify v0.73.0 delivery).
- **CI-INTEGRATION-VERIFY-01**: Confirm Citus integration and Arrow export integration scripts appear in CI workflows (per v0.75.0 CI-INTEGRATION-01/02). Cross-verification only.
